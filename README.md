# Lygiagretus-egzaminas

Repository for parallel/concurrent programming exam tasks (“Lygiagretus” = parallel).  
The codebase is multi-language, primarily **Python** and **C++**, with supporting **C**, **Shell/Batch**, and **CMake**.

## What’s inside

Typical contents you may find in this repo:

- **Python implementations** of exam exercises and/or helpers/scripts.
- **C/C++ implementations** of parallel algorithms or concurrency exercises.
- **CMake build files** for native code.
- **Shell/Batch scripts** to build, run, or automate tests on Linux/macOS and Windows.

### Requirements

- Python 3.10+ recommended
- A C/C++ compiler (GCC/Clang/MSVC)
- CMake 3.16+ recommended
